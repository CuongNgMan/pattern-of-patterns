# Requirements
- NodeJS v12

# How to run?

Install dependencies
```
npm install
```
Start mock-db
```
npm run db
```
Start app
```
npm run start
```

Notes:
- JSON server is listening on port `3030` by default (port can be changed via `json-server.json` file)
- Parcel is listening on port `1234` by default.
- Mock-up data is stored in `db.json` file.
